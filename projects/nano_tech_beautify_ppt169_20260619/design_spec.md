## Visual Language

- **Cards**: Dark `#131926` rectangles with top accent line `#3B82F6`
- **Dividers**: Thin `#1E293B` horizontal rules
- **Section pages**: Full `#0A0E17` background, giant accent-colored number
- **TOC**: Four dark cards in 2×2 grid, each with large number + title
- **Decorations**: Small `#3B82F6` dots, thin geometric lines
- **Icons**: None (text-only deck)
- **Images**: Preserve source images in relevant slides

## Page Structure

Each content slide follows: thin top accent line → slide title → card-based content area → bottom subtle line → page number
