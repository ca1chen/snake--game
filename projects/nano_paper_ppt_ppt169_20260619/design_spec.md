# Design Specification — 纳米技术及纳米机器人的应用和现状

> **kind: free-design**
> **source**: 学科前沿导论1.docx (天津大学 机械工程学院)
> **canvas**: ppt169 (1280×720)
> **slides**: 18 pages
> **style**: 学术答辩风 — 白底深蓝，简洁克制

## Color Scheme

| Role | Hex | Usage |
|------|-----|-------|
| Background | `#FFFFFF` | Content slides |
| Dark bg | `#0B1D3A` | Cover, section dividers, ending |
| Primary | `#003D7C` | Titles, accents (天大字色) |
| Accent | `#0066B3` | Cards, highlights |
| Text dark | `#1A1A1A` | Body text |
| Text gray | `#666E75` | Secondary text |
| Card bg | `#F4F6F9` | Content card backgrounds |
| Divider | `#E0E4E8` | Thin separators |
| Warm accent | `#D4380D` | Key callouts |

## Typography

| Level | Font | Size | Weight |
|-------|------|------|--------|
| Cover title | Microsoft YaHei | 44px | Bold |
| Section number | Arial | 80px | Bold |
| Section title | Microsoft YaHei | 38px | Bold |
| Slide title | Microsoft YaHei | 28px | Bold |
| Card title | Microsoft YaHei | 17px | Bold |
| Body text | Microsoft YaHei | 14px | Regular |
| Caption | Microsoft YaHei | 11px | Regular |

## Visual Language

- **Cover**: Full dark navy bg, centered title + university branding
- **Section dividers**: Dark bg, large number + vertical bar + title
- **Content slides**: White bg, thin top accent line, card-based layouts
- **Cards**: Light gray `#F4F6F9` rectangles with left accent border
- **Decorations**: Thin lines, minimal geometric shapes
- **Images**: From paper (9 images) in relevant slides
